Update Translations and Voiceovers

It looks like you are updating content for a published curated lesson that contains translations and associated voiceovers in different languages.


Please select an appropriate change for these translations and associated voiceovers:

  • Remove existing translations/voiceovers: Delete all translations and their associated voiceovers for the updated contents. Choose this if your changes are big and the current translations or voiceovers no longer match.

  • Modify existing translations: Select this if your changes mean that existing translations will need to be modified slightly.

    Mark as Stale: Keep the existing translations, but mark them and any English voiceovers for updates. Choose this if your changes are small and only need minor fixes.

  • Leave as is: Keep everything as it is. Choose this if your changes do not affect translations (for example, fixing a small typo).

Note:The updates will only be applied when these changes are saved.
